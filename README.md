## About this Professional Certificate

Data science is one of the hottest professions of the decade, and the demand for data scientists who can analyze data and communicate results to inform data driven decisions has never been greater. This Professional Certificate from IBM will help anyone interested in pursuing a **career in data science** or **machine learning** develop career-relevant skills and experience.

It‚Äôs a myth that to become a data scientist you need a Ph.D. Anyone with a passion for learning can take this Professional Certificate ‚Äì **no prior knowledge of computer science or programming languages required** ‚Äì and develop the skills, tools, and portfolio to have a competitive edge in the job market as an entry level data scientist.

The program consists of 9 online courses that will provide you with the **latest job-ready tools and skills**, including open source tools and libraries, Python, databases, SQL, data visualization, data analysis, statistical analysis, predictive modeling, and machine learning algorithms. You‚Äôll learn data science through hands-on practice in the IBM Cloud using real data science tools and real-world data sets.

Upon successfully completing these courses, you will have built a portfolio of data science projects to provide you with the confidence to plunge into an exciting profession in data science.

In addition to earning a Professional Certificate from Coursera, you'll also receive a **digital badge from IBM** recognizing your proficiency in data science.
